> Tweetable Mathematical Art 习作
> 从Matrix67 blog里面看到了Tweetable Mathematical Art比赛，感觉挺有意思。
> 
> 这个比赛的方法很简单，实现3个函数RD(i, j), GR(i, j), BL(i, j)分别返回像素(i, j)的R/G/B值，最终生成1幅1024*1024的图片，看谁的图片被赞最多。要求是每个函数代码不超过140字符，即Tweetable。

这是凹语言对以上例子的实现。
